Материалы и уход

Публикация и настройки

Данные сайта лежат в файле data.json рядом с index.html. Админка умеет обновлять его в GitHub сама.

Подключение к GitHub

Токен хранится только в этом браузере (localStorage) и никуда, кроме api.github.com, не отправляется. Как создать: GitHub → Settings → Developer settings → Fine-grained tokens → выбрать только этот репозиторий → Permissions → Contents: Read and write.

Без GitHub — вручную

Скачайте обновлённый data.json и загрузите его в репозиторий сами (GitHub → папка с сайтом → Add file → Upload files → заменить data.json).

«Загрузить в админку» — если редактировали на другом устройстве и хотите продолжить здесь.
